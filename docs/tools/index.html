Slack is a communication platform we'll use to discuss specific topics outside of class. It also serves as a resource for students to ask questions and exchange information relevant to the course. You will receive an email with an invitation link to join the course Slack. For personal matters (only) you can email the professor.
R is the statistical programming language we will learn about (albeit superficially) in this class. You can download R here. We will interface with R using RStudio, a fully feautred IDE. RStudio is available to download here.
Praat is a cross-platform, free computer program used for analyzing speech. Students will learn to use Praat to analyze vowels and consonants.
PsychoPy2 is used for presenting stimuli in psycholinguistic experiments. Students will learn to build simple production and perception experiments using pyschopy in builder mode.
